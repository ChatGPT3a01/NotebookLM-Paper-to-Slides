## 📌 專案簡介

本專案是一份 **2 小時教學示範課程**，示範如何運用 AI 工具將英文學術書籍章節轉化為學術簡報。

- **示範書籍**：[The ABCs of How We Learn](https://drive.google.com/file/d/1jLxpYIlTJw-SMQtq2U8BTBFvCvPjGUHj/view?usp=drivesdk)（Schwartz, Tsang & Blair, 2016）
- **示範章節**：Chapter W — Worked Examples（工作範例），第 366–379 頁
- **最終產出**：15 分鐘學術簡報（含投影片內容 + 逐頁口頭講稿）

## 📖 Chapter W 核心概念速查

| 概念 | 英文 | 一句話定義 |
|:---:|:---|:---|
| 工作範例 | Worked Examples | 逐步示範完成程序性任務 |
| 認知負荷理論 | Cognitive Load Theory | 工作記憶容量有限，需減少不必要負荷 |
| 分散注意力效應 | Split-Attention Effect | 圖文分離增加認知負擔 |
| 任務分解 | Task Decomposition | 將複雜問題拆解為子步驟 |
| 專家盲點 | Expert Blind Spot | 專家忘記新手的困難 |
| 自我解釋 | Self-Explanation | 主動思考步驟背後的原因 |
| 逐步淡出 | Fading | 漸進移除範例支持 |
